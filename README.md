# Spotify Customer Churn Prediction

## Project Description
This repository contains our group’s final ML design project where we look at Spotify Customer Churn Prediction. The goal of this project is to build a machine learning system that predicts customer churn using Spotify user behavior data. This project is suppose to support us to make better decisions, help identify at-risk users, and improve customer retention.

## Project Workflow
Our system includes the following main components:
- Schema validation
- Data cleaning and transformation
- Feature engineering
- Data versioning
- Model training
- Experiment tracking and model versioning
- Deployment
- Performance monitoring

## Model Development
Our team compared multiple logistic regression model configurations using MLflow. The final selected model was Model 2, which was chosen because it avoided data leakage while still providing realistic performance and detecting churn users. MLflow was used to track experiments, metrics, artifacts, and model versions throughout the project.
